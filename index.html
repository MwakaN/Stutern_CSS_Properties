CSS and Why It is Important

CSS stands for Cascading Style Sheets It is the coding language that gives a website its look and layout. Along with HTML, CSS is fundamental to web design. Without it, websites would still be plain text on white backgrounds.

CSS allows that fonts other than the default for the browser are specified. It also allows for the color and size of text and links. It also allows for the application of colors to backgrounds.
Webpage elements in boxes are also floated to specific positions on the page So in a nutshell CSS is used to alter the font, color, size, and spacing of the content, how to split it into multiple columns, or add animations and other decorative features.

Margins, Paddings and Borders

Margins, padding and borders are essential elements of spacing when it comes to web design. CSS uses what is called the box model to tell web browsers how to display and space different elements on a web page. All websites are made up of a series of boxes that contain these major components:

Padding - The inner space between the content and the border of your box. Padding works only on elements that have borders. It has no effect on elements that don't have borders. It creates the space between the border and the content of an element.
Border - The perimeter of the box. Borders can be invisible or they could be a thick colored line.
Margin - The outer space (or lack of space) surrounding the box. Margins form the space outside the borders of elements. Margins can still affect an element whether or not it has borders.
